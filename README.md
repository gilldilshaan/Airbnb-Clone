# 🏡 Airbnb Clone

A full stack web application inspired by Airbnb — built from scratch as part of my web development journey.

## ✨ Features
- Browse all property listings
- Create a new listing with images
- Edit and update existing listings
- Delete listings
- Fully responsive design
- MVC architecture for clean code structure

## 🛠️ Tech Stack

| Frontend | Backend | Database |
|----------|---------|----------|
| HTML, CSS | Node.js | MongoDB |
| JavaScript | Express.js | Mongoose |
| EJS Templates | REST APIs | |

## ⚙️ Setup & Installation

```bash
# Clone the repository
git clone https://github.com/gilldilshaan/Airbnb-Clone.git

# Navigate into the folder
cd Airbnb-Clone

# Install dependencies
npm install

# Start the server
node app.js

# Open in browser
http://localhost:3000
```

## 📁 Project Structure

```
airbnb-clone/
├── app.js              → main server file
├── models/
│   └── listing.js      → MongoDB schema
├── views/
│   ├── index.ejs       → all listings page
│   ├── show.ejs        → single listing page
│   ├── new.ejs         → create listing
│   ├── edit.ejs        → edit listing
│   └── layouts/        → boilerplate template
├── public/
│   └── css/style.css   → styling
└── init/
    └── data.js         → sample data
```
